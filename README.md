# MindBridge : 생성형 AI 기반 아동·청소년 정서 분석 및 상담 지원 챗봇 웹 서비스

※ This repository reorganizes my individual contributions from a team project conducted as part of a university course.  
※ 이 repository는 대학교 수업의 일환으로 진행된 팀 프로젝트에서 개인적으로 기여한 부분을 재정리한 것입니다.
________________________________________________________________________
### 프로젝트 배경:   
현대 사회에서 아동·청소년은 다양한 문제 상황에 직면해 있으며 이는 심리적 불안정으로 이어질 수 있기 때문에 조기 발견 및 지속적인 대처가 중요하다.   
그러나 상담 자원 및 인력의 부족으로 인한 시간적·공간적 접근성 문제로 인해 위험 상황에 놓인 아동·청소년을 조기에 발견하여 적절한 상담을 제공하는 것이 어려운 실정이다.  
<br>
### 프로젝트 목표: 
- 아동 문제의 조기 발견 및 개입
- 상담사의 상담 효율 향상
- 지속 가능한 심리상담 제공 
- 상담에 대한 거부감 해소
- 상담 접근성 문제 해결
<br>
### 사용 데이터:
- AI Hub의 「아동·청소년 상담데이터」  
- 만 7~12세 아동 청소년의 약 3,600건의 상담 데이터 중 2,000건의 상담 데이터 사용.  
- 전처리: 원본 데이터에서 주제별로 아동/상담사의 발화를 구분하여 텍스프 파일로 저장 → 전체 텍스트 파일을  하나의 json 파일로 변환.  
<br>
---
### 웹 서비스 소개:   
서비스 사용자는 아동·청소년과 상담사로 구분되며 이는 회원가입 시 선택할 수 있다.   
<br>
#### 아동·청소년 모드
-  챗봇(딥챗봇, 라이트챗봇)과의 상담
- 출석 및 상담을 통한 캐릭터 키우기
- 간단한 그림 맞추기 게임 및 명언 보기
- 자신과 매칭된 상담사와 1대 1 채팅
<br>
#### 상담사 모드
- 자신과 매칭된 아동과 1대 1 채팅
- 자신과 매칭된 아동의 상담별 보고서 확인
- 태그 워드 클라우드, 주제별 위험 수치 Top3 상담 내용 확인
- 사용자-태그 간 네트워크 그래프 확인
